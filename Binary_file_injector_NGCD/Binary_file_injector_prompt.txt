Je souhaite faire un code Matlab qui assure les fonctions suivantes:

- Il scanne un répertoire .\Source_files\ qui contient des binaires, quelle que soient leur extension

- Il cible un fichier particulier dans le répertoire courant appelé "track1.iso" qui contient ces binaires mais sous la forme d'une piste audio, dans un ordre inconnu à cause du stystème de fichiers mais important

- Pour chaque fichier dans .\Source_files\, il extrait les premiers 64 octets, les compare à "track1.iso" pour trouver l'offset de départ de position du fichiers. Si une seule correspondante est trouvée, il injecte le fichier correspondant .\Source_files\ à l'offset correspondant dans "track1.iso".

- Enfin "track1.iso" doit rester inchangé, il est donc copié sous un nouveau nom : "track1_patched.iso".
